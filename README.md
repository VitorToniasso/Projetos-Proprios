# Projetos Proprios

Este repositório contém um projeto simples em HTML, CSS e JavaScript.

## Arquivos

- `index.html` - estrutura da página
- `style.css` - estilos do projeto
- `main.js` - lógica JavaScript

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Edite os arquivos conforme precisar.
